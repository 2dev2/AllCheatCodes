** OC - admin operation
| Name            | Command                                                                                            |
|-----------------+----------------------------------------------------------------------------------------------------|
| Login to server | =oc login -u=<username> -p=<password> --server=<your-openshift-server> --insecure-skip-tls-verify= |
| Check status    | =oc status=                                                                                        |
| List route      | =oc get route=                                                                                     |
** OC - projects & applications
| Name                  | Command                                                                                            |
|-----------------------+----------------------------------------------------------------------------------------------------|
| Create a project      | =oc new-project <projectname> --description=<description> --display-name=<display_name>=           |
| Create an application | =oc new-app openshift/ruby-20-centos7~https://github.com/<your_github_username>/ruby-ex=           |
